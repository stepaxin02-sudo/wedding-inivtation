Готово для GitHub Pages.

Как загрузить:
1. Распакуйте этот ZIP.
2. Загрузите файлы index.html и CNAME в корень репозитория GitHub.
3. Не загружайте ZIP целиком — нужно загрузить именно содержимое архива.
4. В Settings → Pages должен быть выбран Deploy from a branch, main, /(root).

Изменение в этой версии:
- При нажатии «Отправить анкету» форма отправляется через FormSubmit без перехода на страницу FormSubmit.
- После успешной отправки появляется табличка «Анкета отправлена».
